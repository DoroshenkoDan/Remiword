# Remiword

**Remiword** is an app for learning foreign-language vocabulary. Add words to your own
lists, practice them in different quiz modes, and track your progress.

🔗 **Website:** [remiword.com](https://remiword.com)

## Features

- 📝 **Word lists** — add words one by one or paste them in bulk
- 🎯 **Quizzes** — three practice modes: flashcards, multiple choice, and spelling
- 📚 **Grammar** — a reference guide with grammar topics
- 📈 **Progress** — statistics on learned words and practice sessions

## Tech Stack

Next.js 16 · TypeScript · Tailwind CSS · shadcn/ui · Supabase
